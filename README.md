# gold-crawler

tags : `requests`, `pandas`, `beautifulSoup`, `highchart`, `sqlite`

HW4 爬蟲作業為抓取台灣銀行提供的 **黃金買賣價格**，並利用 **sqlite** 進行存取，再使用 **highchart** 繪製圖表，將黃金買賣價格可視化，使投資者更方便看出走勢。

[開發者指南] : 本專案全部依循 PEP8 lint 風格，請務必做過 `flake8`, `black`, `isort`。
